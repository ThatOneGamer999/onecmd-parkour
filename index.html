Parkour Manager in One Command by ThatOneGamer999

Hello! This is a simple guide to using my parkour manager for maps!

Do check out the Wiki for more info.

Setup:

Obtain a command block through /setblock ~ ~ ~ command_block.

Note: You MUST be in creative mode. Next paste in the command you can get from my Repo.

Possible keybinds to do this: 'CTRL-V' 'COMMAND-V' 'STRG-V'.

Next, place a redstone block on one of the SIDES of the command block. A bootiful machine should appear!

To start the machine: Press the button on the GREEN wool. Now: special blocks! To set someone as "In Parkour Mode": '/scoreboard players set parkour 1'

Special blocks:

  • Checkpoint: LIME stained clay ('minecraft:stained_clay:5')
  • Send player back to checkpoint: Grass ('minecraft:grass') or RED stained clay ('minecraft:stained_hardened_clay:14')

Everything is handled automagically by the machine.
